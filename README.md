# 🔐 Secure Coding Review - CodeAlpha Internship

## 📌 Description
This project demonstrates a security review of Python code.

It identifies common vulnerabilities and shows how to fix them.

---

## 🚨 Identified Vulnerabilities

### 1. Hardcoded Credentials
- Username and password are fixed in code
- Risk: Easy to hack

### 2. SQL Injection
- User input directly used in query
- Risk: Attackers can manipulate database

---

## ✅ Secure Coding Improvements

- Use parameterized queries
- Avoid hardcoded credentials
- Validate user input
- Use secure authentication methods

---

## 🛠️ Tools Used
- Python
- Manual Code Review

---

## 🎯 Learning Outcome
- Understanding vulnerabilities
- Secure coding practices
- Risk mitigation techniques
